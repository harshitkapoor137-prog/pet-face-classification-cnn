# 🐶 Pet Face Classification using CNN

A deep learning image classification project developed using **Python**, **TensorFlow**, and **Keras** to classify pet images using a Convolutional Neural Network (CNN).

This project demonstrates the fundamentals of image preprocessing, dataset loading, CNN model development, training, and evaluation for binary image classification.

---

## 🚀 Features

- Image Classification
- TensorFlow Dataset Loader
- Image Normalization
- Convolutional Neural Network (CNN)
- Model Training
- Validation Accuracy
- Accuracy Visualization
- Binary Classification
